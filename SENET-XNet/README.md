### visdom


### tensorBoardX
- 安装
```python
>>> pip install tensorboardX
```
- 使用
```python
 from tensorboardX import SummaryWriter
 writer = SummaryWriter('log')
```
- 添加曲线
```python
 writer.add_scalars('data/group', {'train_loss':loss, 'train_acc':acc}, train_step/100)
```
